
## Setup Environment
- 工程搭建参见 [Ruby on Rails教程](https://railstutorial-china.org/rails4/)
- 要启动工程,运行下列命令:
```
  > bundle install            # 安装必需的gem包
  > rake db:migrate        # 数据库迁移
  > railse run server       # 启动server
```
